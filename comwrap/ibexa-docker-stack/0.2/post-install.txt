  * <fg=blue>Run</> your application:
    1. Create a .env from the .env.dist
    2. Set a project name & the docker-compose config you want to use in your local .env file
    3. Remove the solr service config from your docker-compose file when creating the project for the first time
    4. Run composer install
    5. Create the solr config as described here -> https://doc.ibexa.co/en/latest/search/solr_search_engine/#generating-configuration
    6. Re-add the solr service and build again.
